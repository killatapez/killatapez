## Клонирующийся темплейт для создания множества похожих сайтов на раскрутку рекламы (проект сдан, сайт на этапе запуска)

<img width="1885" height="951" alt="image" src="https://github.com/user-attachments/assets/f72d9083-d940-4840-a010-4f872749cc0d" />

**Проблема:** маркетологам необходим один темплейт сайта с определенным дизайном, который можно склонировать и наполнить
отличающимся контентом (разные бренды на разных сайтах, но каркас сайта по сути один), также на сайте должен быть функционал
конструктора для пополнения и изменения контента вручную  
**Решение:** создать Next.js + React + TinaCMS через автономный режим и хранением GraphQL схем в MongoDB  
**Результат:** создан сайт с JWT авторизацией в админку полностью подстроенной под клиента для редактирования любого элемента,
подключение настроено так чтоб все Docker контейнеры обращались к одному MongoDB инстансу для экономии ресурсов хостинга, а 
также настроен CI/CD по выгрузке обновлений в основной темплейт и клонированию сайтов по нажатию одной кнопки  

## [ААА Моторс](https://aaa-motors.ru)

<img width="1902" height="951" alt="image" src="https://github.com/user-attachments/assets/ecd12639-7751-463d-83a8-06be655d6cc4" />

**Проблема:** сайт компании разрабатывался недобросовестными подрядчиками которые задирали цену и подрывали сроки на самых
простых задачах, необходим человек который сможет поддерживать и развивать сайт  
**Решение:** добавление большого количества нового функционала, например: новые страницы, новые возможности в меню, новые 
страницы в админке, улучшение производительности сайта  
**Результат:** пополнен функционал сайта, значительно улучшен UI, настроен CI/CD (автоматизация выгрузки обновлений которая 
значительно ускорило процесс разработки, выгрузка веток на автоматически созданные поддомены для тестирования), улучшена 
оптимизация сайта (загрузка страницы уменьшена с 2 секунд до 1, показатели здоровья сайта вышли в норму)  

## [Ваш костюм](https://vash-kostum.ru)

<img width="1904" height="952" alt="image" src="https://github.com/user-attachments/assets/ba2feb59-4e23-4cce-84cf-6202b4bb8e00" />


**Проблема:** имеется сайт на легаси технологиях (битрикс проект HTML/CSS/JS), нужно много нововведений  
**Решение:** разработать кучу нового функционала, например: настраиваемый попап с разными причинами вызова, отправкой заявок
на почту и сохранение в админку (также в работе отправка заявок через Telegram API), кастомный слайд со слоем разметки поверх
изображения без изменения общей стилизации старого слайдера, разработка новых страниц по запросу клиента с макетом в Figma  
**Результат:** создал CI/CD пайплайн с хостингом Beget на котором крутится сайт, добавил функционала, освежил монолитный css
файл в scss модули (в работе: редизайн сайта, смена технологического стека и добавление CMS)

## Приложение экрана защиты предприятия (NDA)

<img width="1000" height="593" alt="image" src="https://github.com/user-attachments/assets/01a4b40f-c2c8-4428-b9e9-a70f9346beb2" />

**Проблема:** компания создает визуальную среду для C++ модулей защиты предприятия от угроз, имеется Electron + React приложение  
**Решение:** поддержка и развитие кодовой базы, создание новых экранов с визуалом через Three.js, такие как: фильтруемые таблицы
со списком угроз, роза ветров событий в подсети, карта пошагового перемещения угрозы  
**Результат:** добавил новый функционал, произвел массовый рефакторинг кодовой базы, улучшив быстродействие и загрузку
приложения посредством очистки ненужных ререндеров и углубления нужных ререндеров, улучшил визуал
